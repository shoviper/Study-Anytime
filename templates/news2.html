    K-Engineering World Tour and Workshop 2022

    Welcome, Future Engineers!

    We have good news! This year, we're back again with another open-house workshop. But this year is not the same as the past. Instead of hosting one workshop, we decided to host two! We welcome you to join in this awesome event, whoever you are. More information about the event can be found below!

    Details of the event

    So, as we have stated earlier, two workshops will be hosted this year:

    • - A Python Workshop @ room ECC804 (8th Floor); and
    • - A Web Programming Workshop @ room ECC704 (7th Floor).

    Each workshop will be hosted two times per day across the duration of the event. The schedule is as follows:

    • Round 1: 9:00 a.m. - 12:00 p.m.
    • Round 2: 13:00 p.m. - 16:00 p.m.

    The Registration for both of these workshops is now closed
